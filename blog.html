Question-1: What are the different ways to select an element in the DOM?

Answer:

There are several ways to select an element in the DOM. Some are expressed below.

1. Select an element by it's id.

a. document.getElementById

b. document.querySelector

2. Select an element by it's class name.

a. document.getElementsByClassName

b. document.querySelector

3. Select multiple elements by their common class name.

a. document.getElementsByClassName

b. document.querySelectorAll

4. Select an element by it's tag name.

a. document.getElementsByTagName

b. document.querySelector

5. Select multiple elements by their common tag name.

a. document.getElementsByTagName

b. document.querySelectorAll

6. Select an element by it's name from input field.

a. document.getElementsByName

b. document.querySelector

7. Select multiple element by their common name from input field.

a. document.getElementsByName

b. document.querySelectorAll

8. Select parent element from any of it's children.

a. document.getElementById("childElementId").parentNode

b. document.getElementsByClassName("childElementClassName").parentNode

c. document.querySelector("childElement With Proper ID/ ClassName/ anyAttribute").parentNode

9. Select first child element from parent.

a. document.getElementById("parentElementId").firstChild

b. document.getElementsByClassName("parentElementClassName").firstChild

c. document.querySelector("parentElement With Proper ID/ ClassName/ anyAttribute").firstChild

d. document.getElementById("parentElementId").firstElementChild

e. document.getElementsByClassName("parentElementClassName").firstElementChild

f. document.querySelector("parentElement With Proper ID/ ClassName/ anyAttribute").firstElementChild

10. Select last child element from parent.

a. document.getElementById("parentElementId").lastChild

b. document.getElementsByClassName("parentElementClassName").lastChild

c. document.querySelector("parentElement With Proper ID/ ClassName/ anyAttribute").lastChild

d. document.getElementById("parentElementId").lastElementChild

e. document.getElementsByClassName("parentElementClassName").lastElementChild

f. document.querySelector("parentElement With Proper ID/ ClassName/ anyAttribute").lastElementChild

11. Select next sibling child element from a child element.

a. document.getElementById("currentSiblingId").nextElementSibling

b. document.getElementsByClassName("currentSiblingClassName").nextElementSibling

c. document.querySelector("currentSibling With Proper ID/ ClassName/ anyAttribute").nextElementSibling

12. Select previous sibling child element from a child element.

a. document.getElementById("currentSiblingId").previousElementSibling

b. document.getElementsByClassName("currentSiblingClassName").previousElementSibling

c. document.querySelector("currentSibling With Proper ID/ ClassName/ anyAttribute").previousElementSibling

13. Select all children element from parent.

a. document.getElementById("parentElementId").childNodes

b. document.getElementsByClassName("parentElementClassName").childNodes

c. document.querySelector("parentElement With Proper ID/ ClassName/ anyAttribute").childNodes

d. document.getElementById("parentElementId").children

e. document.getElementsByClassName("parentElementClassName").children

f. document.querySelector("parentElement With Proper ID/ ClassName/ anyAttribute").children

14. Select element body

a. document.body

15. Select elements from CSS selectors.

a. document.querySelector

a. document.querySelectorAll

16. Sometimes elements can be selected dynamically by id, classname, any or multiple attributes.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:

innerHTML
InnerHTML can read both markup tags and contents of an element. When console.log is used for innerHTML of an element, browser displays both HTML tags and contents. Both HTML tags and contents can be set by innerHTML. innerHTML can get hidden markups too.

innerText
InnerText can read only contents of an element and can not get the markup tags. When console.log is used for innerText of an element, browser displays only contents. Only contents can be set by innerText and line-break, white-spaces are included. innerText can not get hidden markups or contents.

textContent
textContent can read only contents of an element and can not get the markup tags. When console.log is used for textContents of an element, browser displays only contents both visible and hidden. Only contents can be set by innerText and line-break, white-spaces are ignored. innerText can get hidden contents.
